# **App Name**: RentFlow

## Core Features:

- Cloud Portfolio Management: Comprehensive property and tenant CRM using Cloud Firestore to manage units, addresses, and individual tenant terms for production-ready reliability.
- Automated Billing Engine: Triggered cloud functions that generate digital invoices on the 1st of every month, complete with automated UPI payment link creation.
- Collections Intelligence Tool: Generative AI analysis tool that identifies potential payment risks and generates custom-toned WhatsApp reminders based on the tenant's payment history.
- Centralized Admin Dashboard: Visual command center displaying high-level stats like total pending rent, monthly trends, and property-wise collection metrics.
- Multi-Channel Communication: Service-oriented notification layer for sending rent reminders and overdue notices via WhatsApp using dynamic template systems.
- Landlord & Super-Admin Portal: Role-based access control with specialized views for single property landlords and portfolio-wide super admins.
- Responsive Billing Views: Mobile-optimized public payment pages for tenants to view pending invoices and initiate deep-link payments from any device.

## Style Guidelines:

- Primary color: Deep Indigo (#283eb0) to convey professionalism and financial stability in a SaaS context.
- Background color: Clean Blue-Grey (#f8f9fb) to maintain a modern, document-centric enterprise feel.
- Accent color: Vibrant Cerulean (#2093df) used sparingly for interactive call-to-action buttons and successful payment indicators.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward look; Body font: 'Inter' (sans-serif) for high legibility across financial tables and reports.
- Clean, card-based interface with heavy use of white space and rounded containers (Material 3 style) to reduce visual cognitive load on complex data views.
- Geometric, thin-line icon set to match the precision and machined aesthetic of the 'Inter' typography.
- Soft state transitions and layout shifts powered by Next.js motion libraries to make navigating through property details feel fluid.
